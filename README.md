# Kaafi Aqua - Water Management System

A comprehensive water refill management system with POS, inventory, and user management.

## Features
- Point of Sale (POS)
- Inventory Management
- Tank Level Monitoring
- Filter Status Tracking
- Expense Tracking
- User Management (Admin/Staff)
- Sales Reports (Excel/PDF export)

## Tech Stack
- **Backend**: Spring Boot 2.7.18, Java 8, MySQL
- **Frontend**: React 18, Vite, Tailwind CSS
- **Security**: JWT Authentication
